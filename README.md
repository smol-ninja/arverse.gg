### To start development

###### Installing dependencies

```bash
yarn install
```

###### Running locally on Port 3000

```bash
yarn dev
```

### Docker deployment

###### Build docker image

```bash
docker-compose build
```

###### Run docker container on Port 8000

```bash
docker-compose up
```

### Production deployment

Auto deployment is enabled via Github workflow.
